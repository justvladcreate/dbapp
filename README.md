# Что это?
Это личный проект приложения для управления базой данных по предметной области *Учёт аренды недвижимости*
Он был создан в качестве выполнения задания для отчёта.

## Как использовать?
Скачиваете релиз, распаковываете архив, запускается .exe

Для работы приложения требуется скачанный PgAdmin 4 и PostgreSQL

После запуска в приложении можно различно управлять созданной структурой БД.
